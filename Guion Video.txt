Hola a todos en este video, les presentaremos nuestro programa desarrollado en Java con Visual Studio, utilizando el paradigma de programación orientado a objetos. Acompáñennos mientras exploramos sus funcionalidades y las fases de construcción que hemos llevado a cabo.

Nuestra aplicación es una solución completa para la gestión de recursos humanos y la liquidación de nóminas. Permite realizar el CRUD (Crear, Leer, Actualizar, Eliminar) de diversas entidades involucradas en el proceso, como lo son los empleados, las EPS, los Fondos de Pensión, las ARLs, las Cajas de Compensación, la  configuración de la empresa y los conceptos de devengo y de deducciones.

La construcción de nuestro software se llevó a cabo en diferentes fases para garantizar su funcionalidad y eficiencia. A continuación voy a detallarles cada una de ellas:

Empezamos con el Análisis y Diseño: En esta etapa inicial, realizamos un análisis exhaustivo de los requisitos del programa. Identificamos las entidades principales involucradas en el proceso y diseñamos la estructura de clases y las relaciones necesarias para implementar el paradigma de programación orientado a objetos. Tambien   desarrollamos una Interfaz Gráfica de Usuario intuitiva y fácil de usar. Esta interfaz permite gestionar todas las funciones del programa de manera sencilla, brindando una experiencia amigable para el usuario.

Despues de esto dimos paso al Desarrollo de Funcionalidades: Una vez completado el análisis y diseño, nos enfocamos en desarrollar las funcionalidades clave del programa. Implementamos las operaciones de gestión CRUD para todas las entidades mencionadas anteriormente. Ademas durante esta fase, desarrollamos los algoritmos necesarios para calcular las cesantías, el interés de cesantías, las primas y las vacaciones. Estas prestaciones se liquidan semestralmente o cuando el empleado se retira de la cooperativa. Continuamos con la Implementación de la funcionalidad para gestionar los conceptos de devengo relacionados con el corte de caña. Dependiendo del tipo de caña (cruda o quemada) y del día (ordinario o festivo) se le asigna a cada empleado un salario específico, al cual se le pueden sumar otros devengos o restar ciertas deducciones. 

Una de las ultimas fases fue la de lograr tener Persistencia de Datos, en este caso implementamos la capacidad de almacenar los datos de forma persistente utilizando archivos de texto o binarios. Esto asegura la integridad y disponibilidad de la información. Para finalizar realizamos pruebas exhaustivas a nuestro programa para asegurarnos de que funcione correctamente y cumpla con todos  los requisitos establecidos.

Conociendo todo lo anterior podemos concluir que este programa representa una solución integral para la gestión de recursos humanos y la liquidación de nóminas. Con la capacidad de gestionar diversas entidades, calcular prestaciones sociales de manera precisa y eficiente, y manejar conceptos de devengo y deducciones de manera optima, nuestro software simplifica y agiliza los procesos relacionados con el personal de tu empresa.Estamos seguros de que nuestro programa se adaptará a las necesidades de tu empresa, ofreciendo una solución completa y confiable para la gestión de recursos humanos y la liquidación de nóminas.
